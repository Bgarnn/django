Ex00: Markdown Cheatsheet

Markdown Syntax

Essential Syntax for writing and formatting text, commonly used for formatting readme files or writing in online forums.

  1. Headings

    • # Heading 1
    • ## Heading 2
    • ### Heading 3
    • #### Heading 4
    • ##### Heading 5
    • ###### Heading 6

  2. Bold and Italic Text

    • **bold text**
    • *italic text*
    • ***bold and italic text***

  3. Lists

    • Unordered List
      • - Item 1
      • - Item 2
        • - Subitem 2.1
        • - Subitem 2.2
    • Ordered List
      1. Item 1
      2. Item 2
        1. Subitem 2.1
        2. Subitem 2.2

  4. Links

    • [Link Text](http://example.com)

  5. Images

    • ![Alt Text](http://example.com/image.jpg)

  6. Blockquotes

    • > This is a blockquote.

  7. Code

    • `Inline code`
    • ```Code Block```

  8. Horizontal Rule

    • ---

  9. Tables

    • | Header 1 | Header 2 |
    • |----------|----------|
    • |   Row 1  |  Data 1  |
    • |   Row 2  |  Data 2  |
